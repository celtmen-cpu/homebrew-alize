# Alizé

Alizé permet de créer des programmes avec une syntaxe simple et un interpréteur dédié.

### Installer l'interpreteur

```bash
brew trust celtmen-cpu/alize
brew tap celtmen-cpu/alize
brew install alize
```

## Utilisation

alize --help donne toutes les commandes
alize --version : affiche la version
alize --update: mise a jour 
alize fichier.alz : interpretation de ce fichier
